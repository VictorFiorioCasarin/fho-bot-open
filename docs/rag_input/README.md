# Arquivos do RAG (privados)

Esta pasta deve receber os arquivos privados usados para alimentar o RAG em ambiente local.

O que colocar aqui:
- PDFs internos da instituição (por exemplo: regulamentos, materiais de apoio e documentos institucionais).
- Arquivos de texto que nao devem ser publicados no repositorio.

Importante:
- Nao publique os PDFs originais no GitHub publico.
- Mantenha apenas esta instrucao no repositório.
- Se for usar esses arquivos no pipeline, adapte o carregamento no RAG para ler esta pasta.
